PepperTux Legacy FAQ (None of these have been asked yet)

What happened to PepperTux v0.1.7?

It was cancelled for mental health reasons. There is a chance it could come back one day, but that'd be after PepperTux Legacy is completed. Also, that version of PepperTux would be named "PepperTux Future"

When will Milestone 1.9 be released?

Not soon, as Vaesea is taking a break from PepperTux development.

What's new in Milestone 1.9?

While there's no Forest World yet, there'll be various improvements to Icy Island and it'll use SuperTux v0.7.0 instead of v0.6.0.

Can I help with Milestone 2?

Yes, you can! You can create art, music and levels. You can also modify this wiki on GitHub to help improve things and keep things updated.

You could also test the mod to make sure things are working and if something isn't, send a bug report on the repository's issues page.

Why is this wiki so basic?

It's meant to be like the oldest SuperTux wiki used from SuperTux v0.3.0 to SuperTux v0.3.2 development. It's also because I'm lazy :3
