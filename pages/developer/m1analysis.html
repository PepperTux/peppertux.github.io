Milestone 1 Analysis

This page looks back to Milestone 1, released all the way back in January 3rd 2026! I still can't believe that it's been that long.

Bad Things

  • A few levels were pretty bad, like Night Mountain or Light At The End.
  • Snow_Background2.png looks really bad and really needs to be fixed. It's clear that it was just a concept for SuperTux in the SuperTux media repository, not meant to be an actual background. This is something that someone else needs to fix, as I can't.
  • Explosions are the 0.6.0 sprites, not 0.5.1 sprites.
  • The mod gets way too difficult at the end. I forgot to remake the bombs on unstable ice part of The Icy Castle. To be fair, it IS the final level of Milestone 1, but it still should be fixed in Milestone 1.9.
  • Arctic Darkness is pretty quiet.
  • Cold Day was never added from PepperTux-Haxe, instead, VOC-Daytime 2 was used.

Good Things

  • The mod was finished in just 2 weeks, which is a very short amount of time. To be fair, there are still some problems (see Bad Things) but I still like it.
  • It's a data-folder-only mod, meaning that it can easily be added to anyone's SuperTux 0.6.0 installation. This can't be done now and it had some problems but it's fine
  • Arctic Darkness is pretty good.
