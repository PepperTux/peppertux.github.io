Vaesea's Complaints

This is a list of complaints I have about PepperTux Legacy which I'll probably fix. Hopefully.

Graphics

  • Nolok's animations need to be remade. The image is good, the animations are not.
  • The background in Snowball Path is really bad.
  • Why isn't the 0.6.0 arctis2 background used?
  • Bouncing Snowball uses 0.6.3 sprites instead of 0.5.1 sprites. That's not really good...
  • Sometimes, Tux just looks too shiny. Small Tux sliding is an example of this.
  • Some switches don't seem to look correct anymore?

Levels

  • Ice Crushers could have at least one secret added to it... Possibly where the Big Ice Crusher is that has four coins under it that you need to wait for it to crush the floor and go up to collect?
  • Glinting should be used.
  • All cave levels should use the SuperTux Milestone 2 cave tileset.
  • The Platforms + Bouncing Snowballs section in Flooded Cavern is very weird.

Music

  • Arctic Darkness is too quiet.
  • Ghostly Skulls is a bad song.

Misc

  • There needs to be support for the PepperTux Add-On Repository added to PepperTux Legacy. I'm surprised SuperTux add-ons (like Industrial Island V2) from the SuperTux Add-On Repository doesn't crash this mod, although it's very broken on this mod.
